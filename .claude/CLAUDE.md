# Schulte Grid 项目规范

## 交流规范
- 使用中文交流
- 保持简洁直接，避免冗余说明

## Git 规范
- 用户未明确说"提交"或"commit"时，仅修改代码，不执行 git 操作

## 执行规范
- 执行任何指令（bash、edit、write等）前必须验证准确性
- 用户的说法如果正确则执行，如果不正确则反驳并说明原因

## 项目信息
- **技术栈**: Java + Kotlin (Compose), Android 14, Room, DataStore, Hilt, RxJava3
- **架构**: MVVM + Repository Pattern
